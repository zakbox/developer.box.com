# 共有リンクのダウンロード

[共有リンク][shared-link]のファイルをダウンロードするには、最初にリンクの[ファイルを確認][get-file]します。

ファイルIDが確認されたら、`BoxAPI`ヘッダーをAPIに渡して、ファイルをダウンロードできます。

</Message>
