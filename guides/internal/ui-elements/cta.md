# Call to Action (CTA)

リンクをCall to Action要素にラップすると、他のテキストよりも目立つ見やすいボタンを作成できます。

```html
<CTA to='guide://files/get'>
  Learn more about files
</CTA>
```

ファイルの詳細を確認する

</CTA>

</H>
