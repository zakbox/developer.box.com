# グリッド

基本的なグリッドは、要素を`Grid`要素にラップすることで作成できます。2列、3列、4列のグリッドがサポートされています。

```html
<Grid columns='4'>
  ...
</Grid>
```

<Message warning>

グリッドのコンテンツに応じて、返される結果が異なる場合があります。ネストされたカスタム要素と組み合わせてグリッドを使用することをお勧めします。

</Message>
